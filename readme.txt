A plugin to enable the simple addition of CMF Ads advertising code as a Wordpress Widget.

== Description ==

<a href="http://www.cmfads.com">CMF Ads</a> provides low-cost advertising with high benefits for publishers

This plugin makes adding the advertising code to a Wordpress sidebar extremely simple and makes it easy to add or remove extra ad spaces with the change of a dropdown.

Any bugs with this plugin should be reported on <a href="https://bugzilla.geekyhabitat.com">GeekyHabitat Bugzilla</a>

== Installation ==

To install the widget follow the instructions below:

1. Download the plugin and then extract to your wp-content/plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. On the Design --> Widgets page, drop the widget onto your Sidebar
1. Click 'Edit' and enter the following details
1. Enter the title you want to display for the CMF Ads widget
1. Enter your CMF Site ID.
1. Select the Widget Size you want.
1. Select if you wish to display the CMF Ad Link Text below your widget.
1. Select if you wish to centre align the widget.
1. Click save and the widget will appear on your blog.
